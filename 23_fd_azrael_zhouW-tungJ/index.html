    The fastest way to start using the Foundation features showcased on this site is to simply add the CDN link into your HTML head section.

    Alternatively, download the complete version here for all the features available in the framework.

    Grid System

    Starting from full screen, try resizing the window so that it gets smaller and smaller. The number after medium denotes what they do when the window is medium sized. And the number after small denotes what they do when the window is small sized. See line 52-68.

    Buttons

    Coloring

    Options: primary, secondary, success, alert, warning
    Changes the button's color based on the class of button that is specified. See line 68-72

    Sizing

    Options: tiny, small, large, expanded
    Changes the button's size based on the class of button that is specified. Expanded will change the width of the button to fill the entire container. See line 80-84

    Others

    Options: hollow, clear
    Hollow creates a button without a background but with a border.
    Clear creates a button without a background nor a border.
    Disabled creates a button that looks like it is disabled; the disabled class must be added to actually disable it. See line 92-94

    Any combination of these classes can also be created: top

    Callouts

    Callouts are a type of container that creates a box and has styling features. They also have the color and sizing options that can be found with buttons except tiny, expanded, hollow and clear. See line 110-130
